# 融智助手 产品功能规格

## 一、产品定位

**融智助手**是一个面向小微企业主和个体创业者的融资智能平台，通过专业的融资知识和AI技术，帮助用户更高效地获取融资。

### 核心价值
- 📚 **专业内容**：从银行客户经理视角分享实战融资知识
- 🤖 **AI赋能**：智能融资顾问，随时解答融资问题
- 📊 **实用工具**：贷款计算器、资质评估、方案对比

### 目标用户
- 小微企业主
- 个体工商户
- 自由职业者
- 创业者

### 2.2 内容系统

#### 内容分类
1. **融资基础**：融资入门知识、常见问题

#### 内容类型
- 📄 文章：图文内容
- 📊 工具：计算器、模板
- 🎥 视频：课程视频（二期）

#### 会员权限
- 普通用户：免费内容 + 部分预览
- 付费会员：全部内容
- 超级会员：全部内容 + 可下载

---

### 2.3 AI融资顾问

#### 功能描述
基于大语言模型的智能融资顾问，可以回答用户的融资相关问题。

#### 对话场景
1. **贷款咨询**
   - "我是个体户，想申请贷款，需要什么条件？"
   - "小微企业贷款哪家银行利率最低？"

2. **资质评估**
   - "我的征信有逾期记录，还能贷款吗？"
   - "个体户没有流水可以贷款吗？"

3. **方案推荐**
   - "我需要30万周转资金，有什么推荐方案？"
   - "想开个小店，可以申请创业贷款吗？"

#### 积分消耗
- 普通用户：无权限
- 付费会员：每月100次
- 超级会员：无限次

---

### 2.4 贷款计算器

#### 计算类型
1. **等额本息计算器**
   - 输入：贷款金额、年利率、期限
   - 输出：月供、总利息、还款总额

2. **等额本金计算器**
   - 输入：贷款金额、年利率、期限
   - 输出：首月月供、末月月供、总利息

3. **利率对比工具**
   - 对比不同银行的利率差异

#### 使用权限
- 所有用户均可使用
- 付费会员可保存计算结果

## 四、技术架构

### 前端架构
```
Vue 3 + Vite + Element Plus
├── 用户端 Web
├── 管理后台 Web
└── 移动端适配
```

### 后端架构
```
CloudBase 云函数
├── user - 用户服务
├── article - 内容服务
├── payment - 支付服务
├── ai - AI服务
└── admin - 管理服务
```

### 数据存储
```
MySQL - 结构化数据
├── 用户数据
├── 订单数据
├── 文章数据
└── 产品数据

NoSQL - 非结构化数据
├── AI对话记录
├── 用户行为日志
└── 系统配置
```
