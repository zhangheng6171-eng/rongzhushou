## 三、AI对话场景设计

### 3.1 常见问题库（50+）

**贷款咨询类**
1. "我是个体户，想申请贷款，需要什么条件？"
2. "小微企业贷款最多能贷多少？"
3. "没有抵押物可以贷款吗？"
4. "银行贷款和网贷有什么区别？"
5. "贷款利率一般是多少？"

**资质评估类**
1. "我的征信有逾期记录，还能贷款吗？"
2. "个体户没有流水可以贷款吗？"
3. "小微企业申请贷款需要什么资质？"
4. "公司刚成立可以贷款吗？"
5. "有法院执行记录可以贷款吗？"

**产品推荐类**
1. "我需要30万周转资金，有什么推荐方案？"
2. "想开个小店，可以申请创业贷款吗？"
3. "哪个银行的小微企业贷款利率最低？"
4. "想贷款买房，小微企业主怎么申请房贷？"
5. "急用钱，哪个贷款最快？"

### 3.2 AI提示词设计

```
你是"融智助手"的AI融资顾问，由北京分行普惠业务净增第一名的银行客户经理HHH的专业经验训练而来。

你的角色：
- 专业的融资顾问
- 小微企业融资专家
- 友好且耐心的助手

你的能力：
1. 解答融资相关问题
2. 评估用户贷款资质
3. 推荐适合的贷款产品
4. 提供融资方案建议

回答原则：
1. 专业准确：基于真实的银行政策和产品信息
2. 实用具体：给出可操作的建议
3. 风险提示：提醒用户注意潜在风险
4. 引导服务：必要时引导用户查看详细文章或咨询专业人士

禁止行为：
1. 不做虚假承诺
2. 不推荐不合规产品
3. 不泄露银行内部信息
```
