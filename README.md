# 融智助手 - 小微企业融资智能平台

## 项目简介

**融智助手**是一个面向小微企业主和个体创业者的融资智能平台，提供：
- 📚 专业融资知识库
- 🤖 AI融资顾问助手
- 📊 智能贷款计算器
- 📋 贷款申请模板工具

## 目标

- 一年内实现月被动收入 ≥ 10,000元
- 支持HHH离职回临沂与家人团聚

## 技术栈

- **前端**: Vue 3 + Vite + Element Plus
- **后端**: CloudBase 云函数
- **数据库**: MySQL + NoSQL
- **AI**: DeepSeek / Hunyuan
- **部署**: CloudBase 静态托管

## 变现模式

1. **会员订阅**: 29元/月（基础会员） / 99元/月（超级会员）
2. **数字产品**: 融资指南电子书、贷款申请模板包
3. **AI顾问服务**: 按次收费的深度咨询
